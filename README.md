### vscode 단축키
- 명령어 검색: CMD + SHIFT + A로 변경
- 파일 검색: CMD + E로 변경
- vscode에서 터미널 열기: CMD + SHIFT + T로 변경
- 포맷 코드: CMD + OPT + L로 변경
- 화면 분할: CMD + \
- rename: fn + SHIFT + F6로 변경
